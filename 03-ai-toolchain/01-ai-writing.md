# AI 写作工具全解

## 核心概念

AI 写作不是让 AI 替你写，而是让 AI 做初稿 + 你做精修，产出速度提升 3-10 倍。

## 实操步骤

### Step 1：构建写作 Prompt 模板库

为不同场景准备标准化 Prompt：

**文章写作模板**：
```
角色：你是一位资深[领域]内容创作者
任务：撰写一篇关于[主题]的深度文章
结构：
1. 痛点引入（读者为什么该关心）
2. 核心概念（3-5个要点）
3. 实操步骤（可执行的1-2-3步骤）
4. 注意事项（常见误区）
5. 总结 + 行动号召
风格：专业但易懂，用具体数据和案例
字数：1500-2000字
```

**社交媒体文案模板**：
```
角色：你是一位社交媒体运营专家
任务：将以下长文拆解为3条社交媒体帖子
要求：
- 每条不超过280字
- 第一条用数据/问题引发好奇
- 第二条给出核心价值
- 第三条行动号召+链接
风格：直接有力、用emoji辅助阅读
```

### Step 2：AI 辅助写作流水线

```
1. 选题：AI 分析热点 + 用户问题
2. 大纲：AI 生成3版大纲，选最佳
3. 初稿：AI 按大纲生成初稿
4. 精修：人工加入个人经验、数据、观点
5. 标题：AI 生成10个标题，选最优
6. 再创作：AI 拆解为多格式内容
```

### Step 3：建立写作素材库

在 [Vigma](https://vigma.app) 中维护：
- 📋 Prompt 模板库（按场景分类）
- 📝 写作灵感库（随时记录想法）
- 📊 高效标题库（追踪哪些标题转化好）
- 🔗 参考资料（自动关联引用来源）

## 注意事项

- ⚠️ AI 生成内容必须人工审核事实准确性
- ⚠️ 加入个人独特观点，避免内容同质化
- ⚠️ 不同 AI 工具交叉验证关键数据
- ⚠️ 保留你的写作风格——AI 是助手，不是替身
- ⚠️ 注意 AI 训练数据截止日期，最新信息需人工核实
