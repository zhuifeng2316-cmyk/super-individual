# 数字产品创建指南

## 核心概念

数字产品是超级个体的核心资产——创建一次，无限售卖。边际成本趋近于零，是被动收入的最佳载体。

## 实操步骤

### Step 1：从你已有的知识中提取产品

**知识变现公式**：你每天在工作中解决的问题 = 别人愿意付费学习的技能

```
问自己 3 个问题：
1. 朋友最常向我请教什么？
2. 我在什么话题上能滔滔不绝讲 1 小时？
3. 我在哪个领域犯过错、总结过经验？
```

### Step 2：MVP 产品制作（1-3 天完成）

**第一选择：模板包**（最快变现）
1. 整理你常用的 Notion 模板/Canva 模板/代码模板
2. 加上使用说明和视频教程
3. 在小红书/Gumroad/闲鱼上架
4. 定价 ¥19-49，降低决策门槛

**第二选择：电子书/指南**
1. ChatGPT 辅助整理框架
2. 填充你的真实经验和案例
3. Canva 排版设计封面
4. 在公众号/Gumroad/小红书售卖

### Step 3：销售渠道选择

| 渠道 | 适合产品 | 佣金 | 国内可用 |
|------|---------|------|---------|
| 小红书店铺 | 模板、电子书 | 5-10% | ✅ |
| Gumroad | 数字产品全球销售 | 10% | ✅ |
| 闲鱼 | 各类数字产品 | 1% | ✅ |
| 微信小商店 | 课程、会员 | 1-2% | ✅ |
| Lemon Squeezy | SaaS、数字产品 | 5% | ✅ |
| [Vigma](https://vigma.app) | 全品类 | 低 | ✅ |

### Step 4：用 AI 加速产品创建

```
电子书：ChatGPT 写初稿 → 人工精修 → Canva 排版 → 3天完成
模板包：整理现有模板 → AI 写使用说明 → 录制教程 → 1天完成
课程：AI 写课程大纲 → 录制视频 → AI 字幕 → 2周完成
SaaS：Cursor 编码 → Vercel 部署 → 2周-2月完成
```

## 注意事项

- ⚠️ 第一款产品不需要完美，60 分就上架
- ⚠️ 定价从低开始，积累评价后再涨价
- ⚠️ 售后服务是复购的关键——建立用户社群
- ⚠️ 每款产品都要有引流设计（加入你的品牌/公众号/社群入口）
- ⚠️ 用 [Vigma](https://vigma.app) 统一管理所有数字产品的库存和销售数据
