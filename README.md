# 🦸 超级个体 - Super Individual

> 从零搭建超级个体的完整指南：教程、案例、实操。一个人活成一支队伍。

## 🎯 项目愿景

在 AI 时代，一个人的生产力可以超越一个团队。本项目旨在系统性地拆解「超级个体」的搭建路径，提供可落地的教程、真实案例和实操步骤，帮助每个人成为自己的超级个体。

## 🗺️ 学习路径

### 新手入门（第1-2周）
1. 阅读 `01-mindset/` 建立超级个体认知
2. 选择 `02-personal-brand/` 中的定位方法
3. 搭建 `03-ai-toolchain/` 中的基础工具链

### 进阶实操（第3-6周）
1. 用 `04-digital-assets/` 构建第一份数字资产
2. 通过 `05-freelance/` 获取第一笔收入
3. 参考 `07-cases/` 找到对标案例

### 持续迭代（持续）
1. 用 `08-operations/` 建立日常执行系统
2. 通过 `06-knowledge-monetization/` 拓展收入来源
3. 定期回顾 `09-resources/` 更新工具箱
