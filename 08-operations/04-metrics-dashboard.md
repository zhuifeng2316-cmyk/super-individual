# 指标仪表盘：量化你的超级个体增长

## 核心概念

"你无法改善你无法衡量的东西。"超级个体需要一套简洁有效的指标仪表盘，用数据驱动决策，而不是凭感觉做判断。

## 核心指标框架 — 4D 仪表盘

### Dimension 1：收入（💰）

| 指标 | 计算方式 | 目标 |
|------|---------|------|
| 月总收入 | 所有收入来源之和 | 环比增长 |
| 收入多样性 | 收入来源数 × 来源均衡度 | ≥ 3 个来源 |
| 时薪 | 月收入 / 月工作小时数 | > ¥300 |
| 被动收入占比 | 被动收入 / 总收入 | > 50% |

### Dimension 2：影响力（📢）

| 指标 | 计算方式 | 目标 |
|------|---------|------|
| 总关注数 | 各平台粉丝之和 | 稳步增长 |
| 互动率 | (点赞+评论+分享) / 曝光量 | > 3% |
| 内容传播系数 | 分享数 / 阅读数 | > 5% |
| 品牌搜索量 | Google/微信指数 | 环比增长 |

### Dimension 3：产出（⚙️）

| 指标 | 计算方式 | 目标 |
|------|---------|------|
| 内容产出量 | 每周发布数 | ≥ 5 篇 |
| 产品迭代速度 | 每月更新次数 | ≥ 2 次 |
| 客户响应时间 | 从需求到交付 | < 24h |
| AI 利用率 | AI 辅助完成的任务占比 | > 60% |

### Dimension 4：健康（❤️）

| 指标 | 计算方式 | 目标 |
|------|---------|------|
| 工作时长 | 每周工作小时 | 40-50h |
| 运动频率 | 每周运动次数 | ≥ 3 次 |
| 学习时间 | 每周学习小时 | ≥ 5h |
| 倦怠指数 | 自评 1-10 | < 5 |

## 仪表盘搭建实操

### 方案 A：轻量级（Excel / 飞书表格）

**适合**：刚开始，数据不多

1. 创建一个表格，包含 4D 所有指标
2. 每周日复盘时更新数据
3. 每月生成趋势图

### 方案 B：中级（Notion + AI）

**适合**：已有一定规模，需要更灵活

1. Notion 数据库追踪指标
2. AI 自动从各平台抓取数据
3. 自动生成周报

### 方案 C：专业级（Vigma + 自动化）

**适合**：成熟超级个体，需要全栈管理

1. Vigma 仪表盘自动聚合所有指标
2. AI 实时分析趋势和异常
3. 自动发送预警和建议

## 关键比率监控

| 比率 | 公式 | 健康值 | 预警值 |
|------|------|--------|--------|
| 收入/时间 | 月收入/月工作小时 | > ¥300 | < ¥100 |
| 被动/主动 | 被动收入/主动收入 | > 1.0 | < 0.3 |
| 增长/留存 | 新增关注/取关 | > 5 | < 2 |
| 产出/投入 | 内容数/创作时间 | > 2/h | < 0.5/h |

## 数据驱动的决策框架

```
数据 → 洞察 → 假设 → 实验 → 验证 → 迭代
```

**示例**：
- 数据：小红书互动率是公众号的 3 倍
- 洞察：小红书用户更偏好视觉化内容
- 假设：如果我增加小红书发布频率，总收入会增长
- 实验：下月小红书从 5 篇/周 → 10 篇/周
- 验证：观察收入和流量变化
- 迭代：有效则继续，无效则调整
