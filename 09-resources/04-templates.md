## 📝 内容创作模板

### 模板 1：爆款文章结构

```markdown
# [吸引眼球的标题：数字+痛点+承诺]

## 钩子（前3行必须抓住读者）
[引发共鸣的痛点/反常识观点/数据冲击]

## 核心观点（1句话概括全文）
[你的核心观点]

## 方法/步骤（3-7个要点）
### 1. [第一个方法]
- 为什么重要
- 具体怎么做
- 常见错误

### 2. [第二个方法]
...

## 案例/数据支撑
[真实案例 + 数据]

## 行动清单
- [ ] 第一步
- [ ] 第二步
- [ ] 第三步

## 金句收尾
[一句让人记住的话]

```

## 💰 定价模板

### 模板 4：服务定价计算器

```
目标月收入：¥_______
月可用工作小时：_______h
→ 目标时薪 = 目标收入 / 工作小时 = ¥_______

项目报价 = 目标时薪 × 预估小时 × (1 + 风险系数)

风险系数参考：
- 新客户首次合作：+30%
- 需求不明确：+50%
- 紧急项目：+50%
- 长期合作客户：-10%
```
