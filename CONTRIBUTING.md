# 🤝 贡献指南

感谢你对「超级个体」项目的关注！欢迎通过以下方式参与：

## 如何贡献

### 1. 内容贡献
- Fork 本仓库
- 在对应模块目录下添加或修改内容
- 提交 Pull Request

### 2. 案例提交
- 在 `07-cases/` 目录下创建新的案例文件
- 请包含：人物背景、方法论、执行步骤、成果数据、经验总结

### 3. 工具推荐
- 在 `09-resources/01-tools.md` 中添加新工具
- 请说明：工具名称、用途、价格、推荐理由

### 4. Issue 反馈
- 内容纠错
- 新主题建议
- 案例推荐

## 内容规范

- 使用 Markdown 格式
- 每篇文章包含：核心概念、实操步骤、注意事项、参考链接
- 案例需注明信息来源
- 保持内容客观，避免过度营销
