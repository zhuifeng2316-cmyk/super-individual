# 财务顾问独立工作室：一个CPA的一人公司实践

她做了12年审计。

四大。

合伙人路线？

不想走。

不是能力问题。

是生活方式问题。

合伙人意味着更多出差。

更多应酬。

更少属于自己的时间。

2024年，她出来做了独立财务顾问。

## 大部分人以为财务顾问靠熟人

他们错了。

熟人是最差的客户。

价格敏感。

期望无限。

欠人情。

她的第一个客户，来自一篇知乎回答。

## 起步策略

她在知乎上回答了50个关于"小微企业财税"的问题。

不是泛泛而谈。

每个回答都针对一个具体场景：

"年营收500万，怎么合理规划增值税？"

"注册个体户还是有限公司，哪个更省税？"

"老板把个人消费算公司支出，有什么风险？"

三个月后。

每月稳定3-5个咨询。

每个咨询收费2000-5000。

## 她的产品矩阵

很多人以为一人公司只能卖一种东西。

她卖了四种：

**1. 一次性咨询：2000-5000元**

针对具体问题。

比如"我要不要注销这家公司"。

1小时解决问题。

成交快。门槛低。

**2. 月度代账服务：3000-5000元/月**

帮她签约的中小企业做账。

她用AI工具（Vigma的工具链里推荐的几款自动化记账工具）把效率提升了3倍。

一个人同时服务15家企业。

**3. 年度财税规划：2-5万/年**

高客单价。

面向年营收千万以上的企业主。

需要深度的信任和专业的方案。

这是她的利润产品。

**4. 小红书课程：199元**

"小老板必学的10节财税课"。

不靠这个赚钱。

靠它获客。

买了课的人里面，大概10%会升级为咨询客户。

## 收入结构

月均收入：

- 一次性咨询：2万
- 代账服务：4.5万
- 年度规划（分摊到月）：2万
- 课程收入：0.8万

总计约9万。

比在四大高了不止一倍。

关键是：自由。

她搬到了大理。

面朝洱海。

客户全在线上。

## 她做对了什么

Vigma的方法论里有一条我特别喜欢：一人公司的核心不是一个人做所有事，而是一个人做一个完整的商业闭环。

她的闭环是这样的：

小红书课程（获客）→ 一次性咨询（信任建立）→ 月度代账（持续关系）→ 年度规划（高利润升级）。

每一个环节都在为下一个环节输送客户。

这就是飞轮。

一旦转起来，只会越来越快。

## 你能学到的

1. 专业服务最大的障碍不是能力，是获客。用内容打破它。
2. 产品矩阵比单一服务更赚钱。低价获客，高价变现。
3. 地点不重要。互联网让你的专业可以卖到全世界。

如果你有一项专业技能。

你会写。

你会说。

你就有一人公司的全部前提。

剩下的，只是开始。
