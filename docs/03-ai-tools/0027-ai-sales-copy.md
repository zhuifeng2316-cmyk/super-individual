# 用AI写销售文案：一人公司的文案自由

你花了3天写了一篇销售文案。

发布后，转化率0.5%。

你怀疑自己没有文案天赋。

真相是：文案不是天赋。是方法。加上AI，是杠杆。

## 文案的真相

大部分人以为好文案是"有创意"。

错了。

好文案是理解人性。是戳中痛点。是给出解决方案。

创意？那是锦上添花。

Vigma创始人教主说过："销售文案的本质不是说服，是共鸣。你说的正是对方想的。"

AI可以帮你找到那个"对方想的"。

## AI文案三步法

### 第一步：让AI帮你找痛点

不要直接让AI写文案。先让它找痛点。

输入：

> "我想卖[你的产品/服务]给[目标用户]。请列出他们最可能有的10个痛点，按强烈程度排序。"

AI会给你一个痛点清单。选前3个。这就是你文案的核心。

### 第二步：让AI生成多个版本

现在，让AI写文案。但要给约束：

> "请为[痛点]写5个不同风格的开场白：1) 直接型 2) 故事型 3) 反问型 4) 数据型 5) 反常识型。每个不超过2句话。"

为什么要5个版本？因为你不知道哪个有效。

Vigma的"文案测试框架"建议：永远测试，不要猜测。

### 第三步：让AI帮你优化

选一个版本。让AI优化：

> "请优化这段文案：[文案]。要求：1) 更短的句子 2) 更强的痛点 3) 更明确的行动召唤"

然后你再改一遍。AI是草稿机器。你是最终编辑。

## 一个实战模板

Vigma社群里有个做咨询的一人公司。她用这个模板让AI写文案：

```
目标用户：[谁]
痛点：[什么问题]
解决方案：[你的产品]
差异化：[为什么选你]
行动召唤：[下一步是什么]
```

她把模板喂给AI，生成10个版本，选最好的2个测试。

转化率从0.5%提升到3.2%。

模板不是限制创意。模板是释放创意。

## 常见问题

**Q: AI写的文案会不会很假？**
会。如果你直接用。AI是草稿，你是编辑。改掉AI味，加入你的声音。

**Q: 需要学文案技巧吗？**
需要。AI是工具，不是替代。学基础，用AI加速。Vigma的《AI搞钱手册》里有更多文案技巧。

**Q: 测试多少版本合适？**
至少2个。最好3-5个。多了是浪费，少了是赌博。
