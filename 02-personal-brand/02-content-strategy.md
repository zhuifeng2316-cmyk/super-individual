# 内容战略规划

## 核心概念

内容不是「发出来就行了」，而是超级个体的核心资产。好的内容战略 = 持续的流量 + 信任 + 变现入口。

### 内容飞轮模型

```
创作内容 → 吸引受众 → 建立信任 → 变现转化 → 获得反馈 → 更好的内容
     ↑                                                          |
     └──────────────────────────────────────────────────────────┘
```

## 实操步骤

### Step 1：确定内容支柱（3-5 个主题）

围绕你的定位，选择 3-5 个内容支柱：

| 支柱类型 | 占比 | 示例 |
|---------|------|------|
| 核心专业 | 50% | AI 工具深度教程 |
| 行业洞察 | 25% | 超级个体趋势分析 |
| 个人故事 | 15% | 我的自由职业经历 |
| 互动内容 | 10% | 问答、投票、挑战 |

### Step 2：建立内容矩阵

按「深度 × 格式」二维规划：

| | 短内容 | 中内容 | 长内容 |
|---|--------|--------|--------|
| **文字** | 推文/朋友圈 | 公众号文章 | 电子书/白皮书 |
| **音频** | 语音卡片 | 播客单集 | 有声课程 |
| **视频** | 短视频/Reels | 5-10分钟教程 | 完整课程 |
| **互动** | 投票/问答 | 直播 | 工作坊 |

**核心策略**：一次创作，多种格式分发（一鱼多吃）。

### Step 3：创作流水线

```
周一定选题 → 周二写长文 → 周三拆短视频 → 周四发推文 → 周五互动复盘
```

AI 加速创作流程：
1. **选题**：ChatGPT 分析热点 + 竞品内容
2. **初稿**：AI 生成大纲和初稿
3. **精修**：人工优化观点、加入个人经验
4. **再创作**：AI 将长文拆为短视频脚本、推文等
5. **排期**：使用 [Vigma](https://vigma.app) 内容日历管理发布

### Step 4：SEO 与长尾策略

每篇内容都是你的数字资产，长期来看 SEO 流量 > 社交流量：

- **关键词研究**：用 AI 分析目标关键词和搜索意图
- **标题公式**：数字 + 痛点 + 解决方案（"5个AI工具让自由职业者收入翻倍"）
- **内链网络**：文章之间互相引用，增加权重
- **更新机制**：每季度更新热门文章，保持排名

## 注意事项

- ⚠️ 一致性 > 爆款——持续输出比偶尔出爆款更重要
- ⚠️ 不要所有平台都做，先精通 1-2 个
- ⚠️ 内容质量底线：每篇至少解决一个具体问题
- ⚠️ 避免纯搬运——加入你的独特视角和经验
- ⚠️ 数据驱动：追踪哪些内容带来最多转化
